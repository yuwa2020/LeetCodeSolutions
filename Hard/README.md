## Tips for Hard Problems

- Break down the problem into smaller subproblems
- Often requires advanced algorithms or clever optimizations
- Common patterns: advanced DP, complex graph algorithms, segment trees
- May require mathematical insights
- Time complexity can vary widely based on problem constraints
